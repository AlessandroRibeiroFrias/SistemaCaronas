##  Sistema de Caronas  para Eventos de Tecnologia.
### Instalação
- Clonar o projeto
- Realizar a instalaçao do [Composer](https://getcomposer.org/ "Composer")
- Criar a base de dados `sistemacaronas`
- Executar os comandos:
- - `composer install`
- - `php artisan migrate:fresh`
- - `php artisan serve`
